# Pinturas Muniz

Site profissional de serviços de pintura residencial e comercial.

## Funcionalidades

- Design responsivo e moderno
- Menu de navegação interativo
- Seções: Home, Serviços, Galeria, Sobre, Contato
- Formulário de contato
- Animações suaves
- Totalmente otimizado para mobile

## Como visualizar

Abra o arquivo `index.html` no seu navegador ou acesse a versão online.

## Tecnologias

- HTML5
- CSS3
- JavaScript
